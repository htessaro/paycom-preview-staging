Pay.com's risk tools work in layers. You can score transactions before charging
them, block or authenticate charges automatically using rules, act on early fraud
signals before they escalate, and manage the full chargeback lifecycle when
disputes arise. For a conceptual overview of how all tools fit together and when
to use each one, see
[Risk management overview](/docs/paycom-concepts/risk-management-overview).
