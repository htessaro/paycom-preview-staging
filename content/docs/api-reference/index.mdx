The Pay.com API is a REST API organized around predictable, resource-oriented
URLs. It accepts JSON-encoded request bodies, returns JSON-encoded responses,
and uses standard HTTP response codes, authentication, and verbs.

Use the test environment to develop and validate your integration before
connecting to live banking networks. All test requests return `test_mode: true`
in the response body.
