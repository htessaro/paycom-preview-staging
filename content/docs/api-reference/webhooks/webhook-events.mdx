title: Webhook events
description: A complete reference of all webhook event types that Pay.com
  sends, organized by resource.

import { Accordion, Accordions } from 'fumadocs-ui/components/accordion'

Pay.com sends webhook events across the full lifecycle of every resource in
your account. Each event has a `type` field that identifies what happened.
The `data` field contains the full resource object at the time of the event.

A webhook payload has the following shape:

```json
{
  "id": "evt_191439662182903690",
  "resource": "event",
  "type": "charge.succeeded",
  "created": "2022-09-07T13:40:41.751Z",
  "data": { ... }
}
```

Expand a resource below to see the events it can send.
